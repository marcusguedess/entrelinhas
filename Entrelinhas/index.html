Entrelinhas

Ouça, organize e continue seus audiobooks no ponto exato.

O Entrelinhas é um aplicativo web para receber audiobooks, reproduzir seções, salvar progresso automaticamente e guardar marcadores. Para demonstrar a experiência, hospedamos uma amostra de Dom Casmurro pronta para escuta.

O Entrelinhas reproduz seus áudios, mas não cria transcrições. Se você já tiver arquivos .vtt ou .md, importe-os junto com as faixas.

  1. 1. Dê um nome ao livro.
  2. 2. Selecione as faixas de áudio.
  3. 3. Opcionalmente, selecione transcrições com nomes correspondentes.

Para associar arquivos, use o mesmo nome: 01.mp3 + 01.vtt.

          Guia de transcrições

          O texto é opcional e vem de fora do Entrelinhas

          O aplicativo transcreve o áudio?

          Não. O site é hospedado no GitHub Pages e não possui servidor para processar áudio. Seus arquivos permanecem no dispositivo.

          Preciso de uma transcrição?

          Não. O audiobook funciona normalmente sem texto. A transcrição é um recurso complementar para acompanhar ou pesquisar o conteúdo.

          Qual formato devo usar?

          Use .vtt para texto sincronizado com o áudio. Use .md para um texto de leitura sem sincronização temporal.

          Como associar os arquivos?

          Selecione áudio e texto juntos, usando nomes correspondentes:

          01.mp3 + 01.vtt
02.mp3 + 02.md

          Onde consigo o texto?

          Obras legalmente disponíveis podem ter textos em bibliotecas digitais. Um texto integral ainda precisa ser convertido ou sincronizado para virar WebVTT.

          Como gerar uma transcrição?

          Use uma ferramenta externa ou local que exporte VTT. Serviços online recebem seu áudio; confira a política de privacidade antes de usar.

          O importante é manter áudio e texto com nomes correspondentes para que o pareamento seja simples e previsível.
